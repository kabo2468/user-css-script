Copy to Coby

「コピー」を「コビー」にする UserScript

インストール

ページを開いたときだけ動くので、あとから追加された文字(SNSの投稿など)は変わりません。

TrustedHTMLとかの制限で書き換えられないことがあります。

画面が重くなったりするかもしれないので、そのときは無効にしてください。

TamperMonkey の拡張機能ボタンからスクリプトの三角から、開いているサイトを除外できます。

MisskeyMFMHelper

MFMをボタンから入力できるようになる UserScript

インストール

余白をドラッグすると動かすことができます。

注意

@matchにmisskey.ioしか設定してないので、他のインスタンスで使うときはここを編集してください。

バージョンアップしたときに（多分）もとに戻ると思うので、その時はまた編集してください。

MisskeyExpandWidget

Misskey 12.40.0で増えたウィジェットを1つにして広げる UserCSS

インストール

Misskey 12.40.0で増えたウィジェットを1つにして広げるCSSです。
左側のウィジェットが使えなくなります。
右側のウィジェットを編集してください。

後ですること

  • バージョンアップで壊れないようなCSSにする
    • 多分できた
  • もうちょっときれいなCSSにする
